Explicação

  • O sistema se baseia no conceito de uma Memória virtual. Memória Virtual é uma técnica usada pelos sistemas operacionais para expandir a capacidade da memória RAM, permitindo que o sistema execute programas maiores do que a memória física disponível. Ela faz isso armazenando temporariamente partes de programas e dados no disco rígido, enquanto mantém em RAM apenas as partes que estão sendo ativamente usadas. Isso proporciona a ilusão de que o computador tem mais memória do que realmente possui.
  • Cabe ressaltar que a troca de páginas entre disco e memória (swapin) e memória-disco (swapout), e gerênciadas atavés de um algoritimo que tem uma politica própia para saber qual página ele vai desalocar da memória para que uma nova seja alocada, no caso a nova é a página que é requisitada.
  • Neste sistema utilizamos o algoritimo FIFO para demonstrar a troca de páginas entre a memória e o disco rígido. O FIFO adota a lógica de que a primeira página a ser alocada é a primeira a ter que ser desalocada, para que uma nova página que foi solicitada possa ser alocada. Vale dizer que em casos em que a página solicitada já estiver na memória não teremos a atuação do algorítimo.

Passo a passo para utilização:

  1. Digite no campo "Quantas páginas serão simuladas?" a quantidade de páginas que seram alocadas no disco, vale dizer que através destas páginas o sistema gerará uma sequencia de requisicões (de maneira randomica) para a demonstração do algoritimo.
  2. Digite no campo "Quantas requisições teremos?" o número de requisições/interações de solicitação serão feitas em função das páginas. Nisto teremos a quantidade que irá compor nossa sequência de demonstração do algorítimo.
  3. Basta clicar no botão de gerar as requisições e neste clique será gerado a sequência de requisiçoes e também o disco com as páginas alocadas. Um botão surgira nomeado "Requisitar uma nova página".
  4. Agora basta clicar no botão de requisição de uma nova página, a cada clique indica uma interação. Notará que a medida que as páginas são requisitas, elas são retiradas do disco e adicionadas a memória, e o processo inverso também é observado.
